# File Organizer

This project is an LLM-based file organizer that can evaluate and organize any and all files you have, including renaming them.

## Technology Stack

*   **Core Logic**: C# (.NET 8)
*   **User Interface**: Avalonia UI
*   **Goal**: To create a smart, multimodal file organizer that can understand the content of files (images, documents, etc.) and automatically arrange them into a logical folder structure.

## Current Status

The project is in the initial setup phase. The basic solution structure has been created, including:
*   A solution file (`FileOrganizer.sln`)
*   An Avalonia project for the GUI (`FileOrganizer.GUI`)
*   A class library for the core logic (`FileOrganizer.Core`)
